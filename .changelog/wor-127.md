# WOR-127 — Draft Coach Convex module: session queries + mutations

Added the Draft Coach backend module (`convex/draftCoach.ts`), which powers
the private, short-lived AI conversation a party uses to craft a polished
message before sending it into the joint chat. The module provides a session
query, start/send/discard mutations, and a `sendFinalDraft` bridge that posts
the approved draft into the joint chat. All functions enforce auth, party-to-case
authorization, and strict per-user privacy (the other party never sees draft
coach conversations).
