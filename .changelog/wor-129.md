# WOR-129 — Draft Coach Panel: side panel with send gate

Added the Draft Coach Panel UI (`DraftCoachPanel` and `DraftReadyCard`
components). Users can now open a private side panel from the joint chat to
iterate on a message with an AI coach before sending. The panel enforces the
"send gate" — no draft reaches the other party until explicitly approved via
the Send button. Includes mobile bottom-sheet layout, focus management,
accessibility (dialog role, focus trap), and inline error/retry handling.
