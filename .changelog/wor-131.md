# WOR-131 — ClosedCaseView: read-only archive page

Added a permanent read-only archive page for closed cases at
`/cases/:caseId/closed`. When a case reaches a terminal status (Resolved,
Not Resolved, or Abandoned), users see the case header with outcome details,
the full joint chat transcript (no input bar), and tabbed navigation for
their own private coaching and synthesis — the other party's private data is
never exposed. A banner confirms the case is closed and no new messages can
be added.
