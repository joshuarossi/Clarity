## WOR-105 — Shared UI primitives

Added four reusable UI components used across case screens:
**PrivacyBanner** (privacy tint background, lock icon, explanatory modal),
**StatusPill** (four variants: turn/waiting/ready/closed),
**PartyAvatar** (colored initials circle for initiator/invitee/coach), and
**PhaseHeader** (56 px navigation bar with back link, case title, and action slot).
All components support light and dark themes and meet WCAG AA contrast requirements.
