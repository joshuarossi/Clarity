### WOR-124 — Joint chat Convex module

Added the full joint chat backend (`convex/jointChat.ts`) powering real-time
mediation sessions. The module provides a reactive `messages` query for the
live message feed, a `sendUserMessage` mutation that inserts messages and
schedules AI Coach responses, and a complete closure flow — `proposeClosure`,
`confirmClosure`, `rejectClosure`, and `unilateralClose` — that transitions
cases to CLOSED_RESOLVED or CLOSED_UNRESOLVED via the shared state machine.
All functions enforce authentication and party-to-case authorization.
