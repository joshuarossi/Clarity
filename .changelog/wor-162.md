# WOR-162

Scaffolded Tailwind CSS v4 and shadcn/ui infrastructure so that all utility
classes (flex, grid, spacing, colors, responsive breakpoints, etc.) are now
compiled and active at runtime. Added the `@tailwindcss/vite` plugin,
`@import "tailwindcss"` with a `@theme` block mapping existing design tokens
to Tailwind utilities, a `components.json` for the shadcn CLI, and the
`cn()` class-merge helper at `src/lib/utils.ts`.
