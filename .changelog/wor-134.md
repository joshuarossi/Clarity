# WOR-134 — Admin template edit page

Added the `/admin/templates/:id` edit page with a two-pane layout: a form
on the left for editing template content (category, name, global guidance,
coach instructions, draft coach instructions, and notes) and a version
history timeline on the right showing all previously published versions.
Admins can publish new immutable versions or archive a template (with a
confirmation modal displaying the count of affected pinned cases).
