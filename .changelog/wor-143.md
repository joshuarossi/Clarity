## WOR-143 — Admin template-creation UI

Added a dedicated template-creation page (`AdminTemplateCreatePage`) so
that clicking "+ New Template" on `/admin/templates` renders a working
form instead of white-screening. The form captures category, name, global
guidance, coach instructions, and draft coach instructions, then calls
`api.admin.create` to persist the new template with its initial version.
After successful creation the admin is redirected to the new template's
edit page.
