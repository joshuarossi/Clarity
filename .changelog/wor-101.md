## WOR-101 — Transcript compression module

Added `convex/lib/compression.ts`, a server-side utility that keeps
conversation transcripts within AI context-window budgets. When a message
array exceeds its token budget (60 000 for generation, 10 000 for
classification), the module replaces the oldest 50 % of messages with a
single Claude Haiku-generated summary and caches the result by SHA-256
content hash so repeated compressions skip the API call.
