## WOR-102 — App shell (Vite + React + routing + auth)

Added the frontend application shell that all subsequent UI pages build
on. The entry point (`src/main.tsx`) renders React 18 wrapped in
ConvexProvider, AuthProvider, and BrowserRouter. `src/App.tsx` defines
the full route tree — public pages (`/`, `/login`, `/invite/:token`),
protected pages (`/dashboard`, `/cases/*`), and admin-only pages
(`/admin/*`) — with redirect guards for unauthenticated and non-admin
users. A `TopNav` component renders logged-in and case-detail navigation
variants, and a frontend error-handler utility (`src/lib/errorHandler.ts`)
maps all nine Convex error codes to user-friendly toast messages.
