## WOR-102 — App shell (Vite + React + routing + auth)

Added the frontend application shell: Vite dev server, React 18 entry
point with ConvexProvider and AuthProvider, and the full React Router v6
route tree. Protected routes redirect unauthenticated users to `/login`;
admin routes (`/admin/*`) redirect non-admin users to `/dashboard`. The
TopNav component renders logged-in and case-detail variants, and a
frontend error handler maps Convex error codes to user-friendly messages.
