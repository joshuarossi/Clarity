## WOR-103 — Theme and style setup

Added the complete design-token foundation for Clarity's UI. Light and
dark color themes (neutrals, sage accent, lavender coach, party colors,
feedback, tints) are declared as CSS custom properties in
`src/styles/globals.css` with a flash-free dark-mode toggle backed by
`localStorage`. Typography (Inter + JetBrains Mono from Google Fonts),
spacing, radius, shadow, and motion tokens are included alongside a
full set of chat-bubble class recipes (`.cc-bubble` family, 7 variants)
and shadcn/ui Button overrides (primary, secondary, ghost, danger, link)
in `src/styles/components.css`. Animations respect `prefers-reduced-motion`.
