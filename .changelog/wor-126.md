# WOR-126 — JointChatView page: shared conversation with Coach

Added the JointChatView page (`/cases/:caseId/joint`), the real-time shared
conversation space where both parties and the AI Coach interact. Messages
render with party-colored bubbles (initiator / invitee / Coach ⟡ glyph),
timestamps appear on hover, and auto-scroll follows the latest message.
The page includes a "My guidance" panel for reviewing synthesis, a closure
modal, and inline error styling with a Retry button for failed Coach messages.
