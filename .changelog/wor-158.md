# WOR-158

Fixed: CI e2e seed step was failing with "Unauthorized" because it used a
standalone `npx convex run seed:seed` command, which is not authorized under
a preview deploy key. Consolidated the deploy and seed steps into a single
`npx convex deploy --preview-run seed:seed` invocation, which is the
Convex-documented pattern for running internal functions on preview
deployments. This unblocks CI e2e tests.
