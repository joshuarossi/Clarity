## WOR-140

Fixed TypeScript errors in `tests/unit/seed.test.ts` caused by calling
`t.mutation()` with an `internalMutation` export. All 16 call sites now
use `t.run()` with `ctx.runMutation(internal.seed.seed, {})`, which is
the correct convex-test API for invoking internal mutations. The seed
module was also added to `convex/_generated/api.d.ts` so the
`internal.seed.seed` reference resolves correctly.
