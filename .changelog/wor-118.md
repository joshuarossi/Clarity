### WOR-118 — Private coaching AI responses with streaming

The private coaching module now generates real AI responses using Claude
Sonnet with streaming. When a user sends a message, the system assembles
a privacy-isolated prompt (PRIVATE_COACH role — the other party's messages
are never included), streams tokens back into the database in real time,
and marks the message complete with a token count. Failures are retried
once (with exponential backoff for rate limits) before the message is
marked as an error.
