# WOR-128 — Draft Coach AI action: generateResponse with readiness detection

Added the AI generation backend for the Draft Coach (`convex/draftCoach.ts`).
The action powers a multi-turn coaching conversation that helps a party compose
a polished joint-chat message. When the user signals readiness (e.g. "I'm
ready", "draft it", or the Generate Draft button), the coach produces a
structured final draft that is stored on the session — but never auto-sent.
Streaming, template-aware prompts, privacy-scoped context (only the drafting
user's synthesis), and automatic retry with error status are all included.
