## WOR-108 — CI pipeline: GitHub Actions (lint, typecheck, unit, E2E)

Added a four-job CI pipeline in `.github/workflows/ci.yml` that triggers
on every push and pull request to `main`. Jobs run sequentially — lint
(ESLint + Prettier), typecheck (`tsc --noEmit`), unit tests (Vitest),
and E2E tests (Playwright with `CLAUDE_MOCK=true`) — so fast-failing
checks gate the slower ones. All jobs use Node.js LTS with npm caching,
and the E2E job uploads Playwright reports as artifacts on failure.
