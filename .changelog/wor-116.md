## WOR-116

Added the **Dashboard** page (`/dashboard`) — the post-login home screen that
shows all of a user's cases at a glance. Cases are split into Active and Closed
sections, each row displaying the other party's name (or "Waiting for invite"),
category, status pill with color+shape+text semantics, last activity time, and
an Enter button. Solo-mode cases are flagged with a distinct badge. The page
includes a "+ New Case" button, an empty-state message for new users, and
skeleton loading placeholders while data is in flight.
