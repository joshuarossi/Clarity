## WOR-111

Added the Cases Convex module (`convex/cases.ts`), the core CRUD layer for
mediation cases. Users can create a new case (standard or solo mode), list
their cases, fetch a single case, view per-party state with privacy
filtering, and update their own intake form fields. Template version is
pinned at creation time, invite tokens are generated for standard cases,
and all endpoints enforce authentication and party-membership checks.
