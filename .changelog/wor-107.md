## WOR-107 — Playwright infrastructure: config, fixtures, Claude mock mode

Added the shared E2E testing foundation under `e2e/`. Playwright is now
configured for Chromium, Firefox, and WebKit with reusable fixtures for
authenticated single-user and two-user browser contexts. A deterministic
Claude mock mode (`CLAUDE_MOCK=true`) lets AI-dependent features run in
tests without hitting the real Anthropic API, with configurable simulated
streaming delay via `CLAUDE_MOCK_DELAY_MS`.
