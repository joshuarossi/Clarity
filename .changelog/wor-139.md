## WOR-139 — Fix typecheck regression in convex/users.ts

Fixed a TypeScript compilation error (TS2345) in the `me` query where
flow-narrowing of `identity.email` did not survive into the
`.withIndex()` callback closure. The narrowed value is now extracted to a
local `const email` before the closure boundary, so `tsc --noEmit`
passes cleanly. Auth-guard behavior is unchanged.
